Analysis Filters

An analysis filter corresponds to a specific sag-tension solution.

Analysis Filter Groups

Analysis filters can be grouped together. This allows sag-tension results to be linked to a specific purpose. Examples of analysis groups could be:

  • Loads for Structure Design
  • Maximum Sag for Clearance Checks
  • Stringing Tensions

Analysis Filter Manager

Analysis filters can be managed from the Edit menu. Groups can be added, removed, or rearranged. The same operations can also be applied to the filters within the groups.
