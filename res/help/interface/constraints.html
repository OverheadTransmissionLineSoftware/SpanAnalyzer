Constraints

A cable constraint is a limit that the cable should not exceed. In SpanAnalyzer, a cable constraint does not impose the limits on the span or affect the sag-tension results. Constraints are instead displayed in a report that can be checked by the user.

Cable constraints can be set to apply to a specific cable, or to all cables if the entry is blank. The constraints displayed in the report are filtered to display only the relevant constraints.

Cable Constraint Manager

The cable constraint manager is launched from the Edit menu, and shows all of the constraints used in the application. The manager can add, remove, edit, or change the ordering of constraints in the list.



Cable Constraint Editor

The cable constraint editor is launched when an individual cable constraint needs to be edited.
