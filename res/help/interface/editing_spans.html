Editing Spans

Span data is stored in a document (.spananalyzer) file. All of the span editing starts from the Edit Pane. Spans can be added, removed, copied, edited, or moved within the list.

To trigger an editing action, either:

  • Select a span and click a button in the edit pane.
  • Right click a span and select a menu entry.

Span Editor

The span editor is launched when an individual span needs to be edited.




Cable

The cable to use. All of the cables in the application data are displayed.

Constraint Limit

The constraint limit is the primary factor for setting the span tension. There are several methods to define the limit, but all are tied to a specific weathercase and condition.

Stretch

Stretch can be determined two different ways: heavy load and creep. There are weather cases that can be selected for each stretch type.

Span Type

The span type determines the which span geometry options are available.

  • Dead-end - Indicates that there is only a single span between dead-end structures. Only the line section geometry is used.
  • Ruling Span - Indicates that there are several spans between dead-end structures. Both the line section and span geometry can be used.

Geometry

The Line Section Geometry represents the average geometry of all the spans. Usually the average is weighted to favor longer spans.

The Span Geometry represents the geometry of a single span within a line section.



  • Horizontal Spacing - The horizontal spacing between span ends.
  • Vertical Spacing - The vertical spacing between span ends. Positive values indicate the ahead-on-line end is higher than the back-on-line end.
