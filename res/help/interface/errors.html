Errors

If any errors are encountered, a single pop-up will be displayed. To see the specific error message, check the Log Window.

Troubleshooting

The error message should get you pointed in the right direction, but if it's not helpful try the following:

  • Check the span parameters (constraint, geometry, stretch, etc).
  • Check the weathercases.
  • Check the cable model. The cable model needs to be stable enough to use for a sag-tension solution.
  • Restart the program.
  • Sleep on it.
  • Get some technical support.
