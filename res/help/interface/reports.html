Reports

The analysis data can be viewed in the Report Window.



Report Type

The report types break up the sag-tension solution data into categories. All reports show the weather case and condition in the first two columns, followed by the calculated data.

Constraint

This report shows the applicable cable constraints.

  • Type = constraint type
  • Limit = limit
  • Actual = actual value
  • Usage = percent usage of the limit

Sag-Tension

This report shows the typical sag-tension data.

  • Wv = vertical unit weight
  • Wt = transverse unit weight
  • Wr = resultant unit weight
  • H = catenary horizontal tension
  • H/w = catenary constant

Tension Distribution

This report shows the tension in each cable component (if applicable).

  • Hs = shell horizontal tension
  • Hc = core horizontal tension

Catenary - Along Curve

This report shows catenary data that is defined along the curve.

  • H = horizontal tension
  • w = unit weight
  • H/w = constant
  • Sag
  • L = length
  • Ls = slack
  • Swing

Catenary - Endpoints

This report shows catenary data at the left and right end points.

  • Ts = left point support tension
  • Tv = left point vertical tension
  • A = left point angle
  • blank
  • Ts = right point support tension
  • Tv = right point vertical tension
  • A = right point angle

Length

This report shows cable length.

  • Lu = unloaded length
  • Ll = loaded length

Filters

The analysis filters groups are used to select the sag-tension solutions that are displayed in the table.

Sorting

The results shown in the table can be sorted. Clicking on a column will trigger a sort, and clicking again will alternate ascending/descending.

To clear sorting, right click on any column and select 'Clear Sort'.

Exporting

Currently, the only way to export the results is copy/paste. Right click on the table columns or rows and select the copy option that fits best.
