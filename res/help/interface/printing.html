Printing

SpanAnalyzer supports printing the active plot. Get the plot oriented, then capture the view with the printer.

Page Setup

Look over the page setup for size and layout before printing. This can be launched from the File menu.

Preview / Print

Once you are ready to print, look in the File menu. A preview can be generated before printing if needed.

During printing the background color is converted to white. This keeps ink from being wasted. Neutral line colors that show up well on white and dark backgrounds may be best if printing is used heavily.
