Editing Weathercases

As mentioned in the Application Files section, weathercase data is stored in the SpanAnalyzer Data file.

Weathercase Manager

The weathercase manager is launched from the Edit menu, and shows all of the weathercases used in the application. The manager can add, remove, create new, edit, or change the ordering of weathercases in the list.



Weathercase Editor

The weathercase editor is launched when a weathercase is double clicked.
