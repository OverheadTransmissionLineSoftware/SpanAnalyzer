Introduction

SpanAnalyzer is a program developed by Overhead Transmission Line Software (OTLS). It was built to provide transmission line engineers a flexible, open-source alternative for sag-tension calculations. The state of the current industry software is proprietary, and the calculation methods are left unexplained.

Some of the engineering/calculation features include:

  • Single span sag-tension
  • Inclined geometry
  • Experimental Plastic Elongation (EPE) Cable Model
  • Load and creep based stretch

Some of the user interface features include:

  • Load/save files with multiple span configurations
  • Undo/redo actions
  • Multi-threaded analysis
  • Customizable reports
  • Plotting
  • Printing
